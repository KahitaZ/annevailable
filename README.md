# annevailable

Birthday gift website for DJ Annevailable. Single-page digital business card hosted on GitHub Pages at [annevailable.com](https://annevailable.com).

## Color Palette

| Role | Name | Hex |
|---|---|---|
| Background | Near-black | `#0e0e0e` |
| Primary / Logo | Living Coral | `#FF6B50` |
| Fluid accent | Electric Teal | `#0ABAB5` |
| Text | Off-white | `#f0ebe6` |

Coral and teal are complementary opposites on the color wheel — intentional pairing.

## Versions

- `index.html` — current version (WebGL2 fluid simulation, teal disturbance)
- `index_coral.html` — previous version (Three.js + UnrealBloom coral spark effect)
